# Tabify

Comma-delimited list prettifier.

## Description

Tabify converts comma-delimited lists into readable text.

## Sample input

```
Year,Make,Model
1997,Ford,E350
2000,Mercury,Cougar

, => |
```

## Sample output

```
Year | Make    | Model
1997 | Ford    | E350
2000 | Mercury | Cougar
```
